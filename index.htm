AnimateScroll + Animate.css – Options Demo

Scroll down slowly to see how AnimateScroll ties classic Animate.css effects to scroll-driven view timelines. Each block shows one configuration so differences remain visible on all screen sizes.

Basic animatescroll + entry

Simple scroll in from bottom

Uses .animatescroll with .as-range-entry and the default range. The animation runs while the element is entering the viewport.

Classes: animate__animated animate__backInUp animatescroll as-range-entry

Alias .scroll-animate

Using the alias class

Demonstrates the semantic alias .scroll-animate. Behaviour is identical to .animatescroll.

Classes: animate__animated animate__backInRight scroll-animate as-range-entry

Range entry

Range: entry

The animation runs while the element enters the viewport. Scroll up and down to see how the progress follows the scroll direction.

Classes: as-range-entry

Range cover

Range: cover

The animation is driven while the element moves through the viewport from entering to leaving.

Classes: as-range-cover

Range contain

Range: contain

The animation only runs while the element is fully contained in the viewport. It starts a bit later and finishes earlier.

Classes: as-range-contain

Offset early soft

Offset: early (soft)

The animation is compressed into the first part of the entry phase. It feels like it finishes a bit earlier while the element is still entering.

Classes: as-range-entry as-offset-early-soft

Offset early strong

Offset: early (strong)

Stronger early behaviour: the animation completes in the first slice of the entry phase, so most of the movement happens quickly as the element comes into view.

Classes: as-range-entry as-offset-early-strong

Offset late soft

Offset: late (soft)

The animation starts a bit later in the entry phase. The element is partially visible before the motion really kicks in.

Classes: as-range-entry as-offset-late-soft

Offset late strong + delay

Offset: late (strong) with Animate.css delay

Combines a clearly delayed animation range with the built-in animate__delay-1s class from Animate.css.

Classes: as-range-entry as-offset-late-strong animate__delay-1s

Custom inline range

Custom inline animation range

This block overrides --as-range inline to fine tune the start and end of the scroll-driven animation. The movement is distributed between 15% and 85% of the entry phase.

Classes: Inline style: --as-range: entry 15% 85%;

Note: If animation-timeline: view() is not supported or the user prefers reduced motion, all elements will simply appear without scroll-driven animation.
